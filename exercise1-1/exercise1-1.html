Instructions

The goal of this exercise is to create a button that will add a shape to the canvas when it is clicked. The button should take up the full width of the canvas and sit at the bottom of the canvas.

Here are the steps to add a button with a click event listener:

  1. Create a global variable to store the button once it is created but don't initialise it.
  2. After createCanvas() has been called, use the createButton() function to create a new button. Store the button in the global variable created in step 1. If you run your code now, you should see the button on the page but it won't be where it's supposed to be.
  3. To correctly position the button, it needs to be moved inside the canvas.
    1. Select the <main> HTML element that contains the canvas: 
      const container = select("main");
    2. Make the <main> HTML the parent of the button (replace btn with the name of your button variable): 
      btn.parent(container);
  4. Use the p5.Element size() and position() methods to make the button as wide as the canvas and appear at the bottom.
  5. Write a function that will add a new shape to the canvas when it is called. The function should have no parameters and no return. This will be your custom event handler.
  6. Use the p5.Element mouseClicked() method to connect your event handler to the button's mouse click event listener.

When you are finished, go to check-exercise1-1.html to check your work.

Expected output:

You will find it helpful to have the examples from last lecture open as there are many similarities.

If you're having trouble making the button click call your event handler function, make sure you pass only the name of your event handler function (no brackets) to the button's mouseClicked() method.
