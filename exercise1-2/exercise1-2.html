Exercise 1.2

Your sketch:

Instructions

Create a (very) basic theme selector that allows the user to select a background colour and a fill colour. Take a look at the expected output video below.

Create a canvas in your preferred size. Draw a shape or some text on the canvas but don't set any colours yet.

Add two colour pickers using the createColorPicker() function. Refer to the documentation to find out what parameters the function expects. Don't forget to save both colour pickers to global variables.

The steps for positioning colour pickers are exactly the same as for buttons:

  1. After the colour pickers are created and saved to global variables, select the <main> element. You only need to do this once per sketch:
    const container = select("main");
  2. Make the <main> element the parent of both colour pickers (replace colourPicker1 and colourPicker2 with your variable names):
    colourPicker1.parent(container);
colourPicker2.parent(container);
  3. Position the colour pickers wherever you like:
    colourPicker1.position(0, 100);
colourPicker2.position(width / 2, 100);

Finally, use the value() method to get the currently selected colour of each colour picker and use them to set the background and fill colours of your sketch:

background(colourPicker1.value());
fill(colourPicker2.value());

When you are finished, go to check-exercise1-2.html to check your work.

Expected output:

You will find it helpful to have the examples from last lecture open as there are many similarities.

You don't need to add an event listener to the colour pickers.

You also don't need to match the layout in the video or add labels to the colour pickers (unless you want to test your knowledge of drawing with text).
